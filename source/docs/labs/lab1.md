# Лабораторная работа 1

**Тема:** Создание и развертывание статического сайта на базе MkDocs с публикацией на GitHub Pages

## Цель работы

Освоить процесс создания статического сайта с использованием генератора документации MkDocs и научиться организовывать структуру документации проекта (портфолио лабораторных работ).

## Задание

Создать сайт-портфолио для размещения выполненных лабораторных работ по курсу "Программирование на Python" с использованием MkDocs и опубликовать его на GitHub Pages.

## Выполнение работы

### 1. Подготовка окружения

Создан репозиторий `healbane.github.io` на GitHub и клонирован на локальный компьютер.

Установлены необходимые инструменты:
```bash
pip install mkdocs-material
```

### 2. Создание структуры проекта

Создана базовая структура сайта:

```
healbane.github.io/
├── docs/
│   ├── index.md           # Главная страница
│   ├── about.md           # Страница "Об авторе"
│   └── labs/              # Раздел лабораторных работ
│       ├── index.md       # Список лабораторных
│       ├── lab1.md        # Лабораторная работа 1
│       └── lab2.md        # Лабораторная работа 2
└── mkdocs.yml             # Конфигурационный файл
```

### 3. Настройка конфигурации

В файле `mkdocs.yml` настроены:
- Тема Material
- Навигация по сайту через параметр `nav`
- Расширения Markdown для подсветки кода
- Поддержка темной и светлой темы

```yaml
site_name: Портфолио
theme:
  name: material
  language: ru
  palette:
    - scheme: slate
      primary: teal
      accent: teal
```

### 4. Создание контента

**Главная страница** (`index.md`):
- Приветствие и описание назначения сайта

**Страница "Об авторе"** (`about.md`):
- Личная информация
- Навыки программирования с иконками
- Контактная информация

**Раздел лабораторных работ**:
- Список выполненных работ
- Отдельные страницы для каждой лабораторной

### 5. Локальная проверка

Запущен локальный сервер для предварительного просмотра:
```bash
mkdocs serve
```

Сайт проверен по адресу `http://127.0.0.1:8000`

### 6. Публикация на GitHub Pages


Сайт опубликован и доступен по адресу: [https://healbane.github.io](https://healbane.github.io)
